## Requirements

- Node.js (recommended 14.x or newer)
- npm or yarn

## Installation

1. Clone the repository:

```powershell
git clone https://github.com/HimanshuRanpariya/app-gemini-brand-checker.git
cd app-gemini-brand-checker
```

2. Install dependencies:

```powershell
npm install
# or
yarn install
```

## Running Locally

Check the `scripts` section in `package.json` and use the appropriate command. Common commands:

```powershell
npm run dev    # start development server
npm start      # start production server (if configured)
npm run build  # build for production
```

If this project is a Next.js app (it contains a `pages/` folder), use:
